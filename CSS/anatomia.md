# Anatomia
```css
h1{
    color: blue;
    font-size: 60px;
    background: gray;
}

h2{
    color: #ffffff;
    font-size: 23px
    background: #000000;
}
```

* Selector - h1, 2
* Declaration - All.
* Properties - color, font-size, background.
* Property Value - blue, 60px, gray / #ffffff, 23px, #000000.
